# 我的笔记本1
## 2020年9月
### 7号 晴转大雨
```
最近已经将近一周没有按规律吃饭了，有的时候一天只有吃一次午饭，晚上熬夜到四五点睡觉，第二天醒来就到了中午了，然后晚饭有时候也不吃了。
我变得看视频，不吃饭，刷手机，一点一点都没有原来的样子。老妈老爸的微信发消息给我，我也不回。我活着这么失败，以后的日子该如何度过。
未来没有期望，没有理想，现在的生活是多么的不堪，真的感觉活着太难了。
对生活，我只想要吃喝，看视频，没有人打扰我，一个人简简单单的活着就行。
```
### 8号 阴转小雨
```
今天早上太累了，下午回到家里睡到了2点多，就吃了一碗泡面和一片面包。感觉在这样子下去会营养不良。身体会配备吃不消的。
学习的日程安排的很好但是我却一天都无法实现下去，每天都是这样的，累。
看了电视剧亚洲美食，讲了菲律宾的一些故事，感觉每个生活在这个世界上的人都一样，有的一家人为了明天能够吃的饱而活着，有的家庭为了吃的
更好而活着，有的家庭为了更好而活着，这些人活着都是为了自己。但是无论如何过日子，日子一天一天的也就过去了，墨守成规的过日子，不断追
求的过日子，总之你想要的日子就是你自己想要的，但是一些日子你可以去最求，但是你去无法去。因为你眼前看到的只是你想要看到的，你最求自
己习惯的事情，一旦做出改变就很难。但是你要明白，你不能一辈子这样下去，我总想着活得不一样，但是不一样真的有那么简单吗。周围的人只要
有工作就好，最求的是早点下班，早点回家休息。简简单单的这样就行，而你呢。
我到现在都在过着自己不知道过着的日子，每天都不知道活着为了什么，感觉自己逐渐的抑郁了。找不到生活的目标了。这样的日子就这样吗。
```
