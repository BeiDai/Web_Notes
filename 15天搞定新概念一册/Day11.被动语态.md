# Day11. 被动语态
## 被动语态的结构
**be+done**
```
窗户被关了。
The window (is closed).被关了。
- 一般现在时的被动 is\am\are + done
他被杀了。 He is killed.

- 一般过去式被动 was\were + done
手表坏了。 The watch is broken.
他昨晚被杀了。 He was killed last night.
她儿子被送到学校去了。 Her son was taken to school.
他的钱包昨天丢了。 Her wallet was lost yesterday.

- 现在完成时的被动 have\has + been + done
他已经被杀了。 He has been killed.
他已经被邀请了两次了。 He has been invited twice.
地面已经被大雪覆盖了。 The ground has been covered with heavy snow.

- 一般将来时的被动 will + be + done.
他将要被杀了。 He will be killed.
这座大楼马上就要建成了。 THe building will be built soon.
决定很快就要做出来了。 THe decision will be made soon.
```
**惯用的被动语态**
```
1.be born...(出生：bear-bore-born)  He was born in 1984.
2.be hurt...(受伤：hurt-hurt-hurt)  I am hurt.
3.be dressed in...(打扮：dress-dressed-dressed)  The lady is dressed in a large funny hat.
```
**何时使用被动语态？**
```
1.不知道动作的发出者。 It is broken.
2.强调动作的发出者。 He breaks it.&It is broken by him.
3.没必要提出动作的发出者。A new building was built.
```
```
The tea (is served).
The seat (is taken).
```
